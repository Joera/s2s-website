Soul2Soul

Websites for humans

Start or join a collectively owned publication with zero risk. Your content stays yours. License it, withdraw it, move it — at any time, no questions asked.

Flat structure, clear roles. Collectives, not hierarchies.

No single entity controls what stays up. Decentralized infrastructure means no platform, no host, no third party can censor or remove your work.

The result is plain HTML — no backend, no framework dependencies. Preservable, archivable, future-proof.

Authors can layer in identity proofs from trusted third-party protocols — so readers know there's a real, accountable person behind the content.

How it works

Author

  • Work from your local Obsidian app
  • License your content to a publication
  • Withdraw your work at any time
  • Content is encrypted and stored on decentralized storage

Publication

  • Collectively owned via a shared wallet
  • All decisions require quorum approval
  • Whitelist authors and assign developers
  • Accept or decline contributions
  • Approve design and template changes

Developer

  • Work from a GitHub repository
  • Preview the website locally
  • Build templates with Handlebars
  • Use custom fields and post types
  • Propose templates and styling for the publication to approve

Site Generator

  • Runs in a distributed, neutral, secure environment — no single party controls it
  • Triggered when an author publishes or withdraws, or a publication approves content
  • The only place where raw content is decrypted — rendered using approved templates
  • Signs the output to guarantee authenticity
  • Uploads the website to decentralized storage as a content tree
  • Records the result on a blockchain — updating the domain to point to the root hash of the website

A domain you can trust

The domain points directly to the content itself. The address isn't a location on a server — it is the content. A blockchain record links the domain to the root hash of the website. Change the content, the address changes. What you see is exactly what was published.

Pages that prove themselves

Every page is cryptographically signed. Proof that the content hasn't been tampered with, that it was written by the author, and licensed to be displayed on exactly this domain and URL. Copy the page anywhere else — the proof won't hold.
